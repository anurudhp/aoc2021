[Advent of Code 2021](https://adventofcode.com/2021)
====================================================

Personal Challenge: Code each solution in a different language.

7. R: `day07.R`
8. Pascal: `day08.pas`

How To Use
----------

Put your input in `inputs/dayXX.in`, then run `make run-dayXX`
